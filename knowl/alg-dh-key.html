Algorithm 3.3.2. Diffie-Hellman key exchange.
  1. (public) Alice and Bob choose a large prime \(p\) and an element \(g\) of large prime order in \(\F_p\ad\text{.}\)
  2. (private) Alice computes \(A = g^a\) and Bob computes \(B = g^b\text{.}\)
  3. (public) Alice and Bob exchange the numbers \(A\) and \(B\text{.}\)
  4. (private) Alice computes \(k = B^a\text{.}\) Bob computes \(k = A^b\text{.}\)
  5. (private) Alice and Bob now possess a shared secret key \(k\text{.}\)
