Suppose that the forcing function is \(F(t)=F_{0}\cos\omega t\text{,}\) so that the governing equation for the system is

\begin{equation*} mu^{\prime\prime}+\gamma u^{\prime}+ku=F(t). \end{equation*}

Since this is a linear equation in \(u\text{,}\) recall that we can write the solution as

\begin{align*} u(t) \amp =c_{1}u_{1}(t)+c_{2}u_{2}(t)+A\cos\omega t+B\sin\omega t\\ \amp =u_{h}+u_{p}, \end{align*}

and it turns out that \(\lim_{t\to\infty}u_{h}(t)=0\text{.}\) (See examples above)
