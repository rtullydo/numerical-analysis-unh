The upshot of all of this is that we can view an equation of the form

\begin{equation*} M \, dx + N \, dy = 0 \end{equation*}

as a differential equation that seeks to find the equipotentials \(f(x,y) = C\) for some unknown function \(f\) with \(\nabla f = \langle M, N \rangle\text{.}\)
