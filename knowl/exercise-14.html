Exercise 1.

Find the general solutions for the following differential equations. Find the explicit solutions if you can. If you can't solve for \(y\) exactly, then leave it as an \emph{implicit} solution:

  1. \(y^{\prime}=ky\) where \(k\) is a parameter.
  2. \({\displaystyle y^{\prime}=\frac{x^{2}}{y}}\)
  3. \({\displaystyle \frac{dy}{dx}=\frac{3x^{2}-1}{3+2y}}\)
  4. \(xy^{\prime}=\frac{\left(1-y^{2}\right)^{1/2}}{y}\)
  5. \({\displaystyle \frac{dy}{dx}=\frac{x^{2}}{1+y^{2}}}\)
  6. \({\displaystyle \frac{dy}{dx}=\frac{x}{\cos\left(y^{2}\right)y}}\)
