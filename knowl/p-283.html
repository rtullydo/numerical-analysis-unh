Let \(F = \langle M(x,y), N(x,y) \rangle\) be a vector field so that the partial derivatives of \(M\) and \(N\) exist and are continuous on a rectangle \(a \leq x \leq b, c \leq y \leq d\text{.}\) Then there exists a potential function \(f\) so that \(\nabla f = F\) if

\begin{equation*} M_y = N_x. \end{equation*}
