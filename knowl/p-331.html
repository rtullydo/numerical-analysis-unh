The operator expression \(D^2 + 3D + 2\) can be factored as symbols into \((D + 2)(D + 1)\) (that this is equal to the original equation applied to \(y\) is a consequence of linearity). Now, using the fact that the derivative of 0 is 0, we assert that the solutions to the equation

\begin{equation*} (D + 1)(D + 2)y = (D + 2)(D+1)y = 0 \end{equation*}

are the solutions to \((D + 1)(y) = 0\) and \((D + 2)y = 0\text{,}\) which are just the first order equations

\begin{gather*} y' + y = 0\\ y' + 2y = 0 \end{gather*}

These are separable, with solutions \(y_1 = c_1 e^{-x}\) and \(y_2 = c_2 e^{-2x}\text{.}\) So the general solution (justified by the principle of superposition, covered in the next section) to the ODE is

\begin{equation*} y = c_1 e^{-x} + c_2 e^{-2x}. \end{equation*}
