Definition 1.1.1.

The standard form of a first order differential equation (that is, an equation with only first derivatives) is

\begin{equation*} \frac{dy}{dt} = f(t,y), \end{equation*}

where \(y = f(t)\) is a function and \(t\) is the independent variable.

An initial value problem (IVP) is a differential equation with an intial condition:

\begin{equation*} \frac{dy}{dt} = f(t, y), \text{ and } y(t_0) = y_0. \end{equation*}
