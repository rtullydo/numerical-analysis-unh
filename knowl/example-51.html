Example 3.3.2.

Find the longest interval in which the solution to the IVP is certain to exist by TheoremÂ 3.3.1:

\begin{equation*} \left(t^{4}-4t^{2}\right)y^{\prime\prime}+\cos ty^{\prime}-e^{t}y=0,\,\,\,\,\,\,y(1)=2,\,\,\,y^{\prime}(1)=1. \end{equation*}

Solution: Rewrite the equation as

\begin{equation*} y^{\prime\prime}+\frac{\cos t}{t^{2}\left(t^{2}-4\right)}y^{\prime}-\frac{e^{t}}{t^{2}\left(t^{2}-4\right)}y=0. \end{equation*}

so that \(p(t)=\frac{\cos t}{t^{2}\left(t^{2}-4\right)}\) and \(q(t)=-\frac{e^{t}}{t^{2}\left(t^{2}-4\right)}\) which are both continuous on \(\left(-\infty,-2\right)\cup\left(-2,0\right)\cup\left(0,2\right)\cup\left(2,\infty\right)\text{.}\) Since \(t_{0}=1\in\left(0,2\right)\) then \(I=\left(0,2\right)\) is the longest interval where \(p(t)\) and \(q(t)\) are both continuous that contains \(t_{0}\text{.}\)
