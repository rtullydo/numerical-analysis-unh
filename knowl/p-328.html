Define the differential operator \(D\) to be the operation of applying \(\frac{d}{dx}\) to a function \(y(x)\text{.}\) The operator \(D\) can be thought of as a function on functions:

\begin{equation*} D(\text{function}) = \text{ derivative}. \end{equation*}

Any order of derivative can be thought of this way. For example,

\begin{equation*} y''' = \frac{d^3 y}{dx^3} = \frac{d^3}{dx^3} y = D^3 y. \end{equation*}

We need a notion of \(D^0\) - what should that mean? It makes sense to set \(D^0(y) = y\) - that is, take no derivatives of \(y\text{.}\) We will use the symbol \(1\) to represent this function, the identity function that leaves \(y\) unchanged.
