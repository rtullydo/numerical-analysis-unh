Theorem 4.2.1.
\(E\)\(\oplus\)
  1. There exists an element \(\mathcal O\) so that
    \begin{equation*} P \oplus \mathcal O = P \end{equation*}
    for all \(P \in E\text{.}\)
  2. For every element \(P\in E\text{,}\) there exists an element \(-P\) so that
    \begin{equation*} P + (-P) = \mathcal O. \end{equation*}
  3. For all \(P, Q, R \in E\text{,}\)
    \begin{equation*} P \oplus (Q \oplus R) = (P \oplus Q) \oplus R. \end{equation*}
  4. For all \(P, Q \in E\text{,}\)
    \begin{equation*} P \oplus Q = Q \oplus P \end{equation*}
