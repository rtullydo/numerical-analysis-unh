Item.

Part (b): Write the differential equation for velocity, when the object has already reched maximum and is already going down..

Solution: Let \(R(v)=v^{2}/5\) be the resistance, then using the above we have

\begin{equation*} m\frac{dv}{dt}=R(v)-mg, \end{equation*}

and we have \(R(v)\) since the object is going down. Thus

\begin{equation*} m\frac{dv}{dt}=\frac{v^{2}}{5}-98. \end{equation*}
