Section 3.2 Second order linear equations

¶

A general second order ODE is of the form

\begin{equation*} \frac{d^{2}y}{dt^{2}}=f\left(t,y,\frac{dy}{dt}\right). \end{equation*}

As is the case with first order equations, we can describe a more structured family of equations. A \(2^{nd}\) order linear ODE is of the form

\begin{equation*} a(t)y^{\prime\prime}+b(t)y^{\prime}+c(t)y=d(t) \end{equation*}

which can be rewritten as

\begin{equation*} y^{\prime\prime}+p(t)y^{\prime}+q(t)y=g(t). \end{equation*}

A 2nd order ODE is called homogeneous if

\begin{equation*} a(t)y^{\prime\prime}+b(t)y^{\prime}+c(t)y=0 \end{equation*}

and nonhomogeneous if

\begin{equation*} a(t)y^{\prime\prime}+b(t)y^{\prime}+c(t)y=d(t) \end{equation*}

for some \(d(t)\) that is NOT identically zero.

An intial value problem for a second order ODE needs to have two initial conditions:

\begin{align*} y(t_{0}) \amp =y_{0},\\ y^{\prime}(t_{0}) \amp =y_{0}^{\prime}. \end{align*}

Subsection 3.2.1 Second order linear homogeneous ODEs with constant coefficients (characteristic equation)

We will begin with the 2nd order linear homogeneous ODEs with constant coefficients that we introduced and motivated in the previous section:

\begin{equation*} ay''+by^{\prime}+cy=0 \end{equation*}

where \(a,b,c\) are real constants.

Consider \(y^{\prime\prime}-y=0\) or

\begin{equation*} y^{\prime\prime}=y. \end{equation*}

Can we think of a solution to this ODE from Calculus 1? A function where its second derivative is equal to itself?

A moment of thought should lead us to conclude that there are two obvious solutions: \(y_{1}(t)=e^{t}\) and \(y_{2}(t)=e^{-t}\text{.}\) But it not hard to check that the larger family of functions \(y = c_{1}e^{t}\) and \(y = c_{2}e^{-t}\) are also solutions for any constants \(c_1, c_2\text{.}\)

Now consider the general ODE

\begin{equation*} ay^{\prime\prime}+by^{\prime}+cy=0. \end{equation*}

Guided by our observation about the solutions to the previous equation, let us assume solutions are of the form \(y(t)=e^{rt}\text{.}\) (If this is an unsatisfying assumption, another appoach will be explained shortly). Then

\begin{align*} y(t) \amp =e^{rt}\\ y^{\prime}(t) \amp =re^{rt}\\ y^{\prime\prime}(t) \amp =r^{2}e^{rt}, \end{align*}

and plugging this into the ODE we have

\begin{align*} LHS=ar^{2}e^{rt}+bre^{rt}+ce^{rt} \amp \overset{?}{=}0\\ e^{rt}\left(ar^{2}+br+c\right) \amp \overset{?}{=}0. \end{align*}

and since \(e^{rt}\neq0\) then

\begin{equation*} ar^{2}+br+c=0. \end{equation*}

Using standard methods for quadratic equations, we can solve for the roots \(r=r_{1},r_{2}\text{.}\)

This is called the characteristic equation of this ODE. If the roots \(r_1, r_2\) are real and distinct, then the general solution to the homogeneous equation is of the form

\begin{equation*} y(t)=c_{1}e^{r_{1}t}+c_{2}e^{r_{2}t}. \end{equation*}

The justification for gluing together the solutions will be presented in the next section as the “principle of superposition”.

Example 3.2.1.

Let's find the general solution of

\begin{equation*} y^{\prime\prime}+5y^{\prime}+6y=0. \end{equation*}
  1. We'll guess that the solution to a solution is \(y(t)=e^{rt}\) for some \(r\text{.}\) Then get
    \begin{equation*} \left(r^{2}+5r+6\right)e^{rt}=0 \end{equation*}
    so that we must have \(r^{2}+5r+6=\left(r+2\right)\left(r+3\right)=0\) so that \(r=-2,-3\text{.}\)
  2. So \(y_{1}(t)=e^{-2t}\) and \(y_{2}(t)=e^{-3t}\) are solutions and
    \begin{equation*} y(t)=c_{1}e^{-2t}+c_{2}e^{-3t} \end{equation*}
    is the general solution.
Example 3.2.2.

Let's find the solution to the following IVP

\begin{equation*} y^{\prime\prime}+5y^{\prime}+6y=0\,\,\,\,y(0)=2,y^{\prime}(0)=-1. \end{equation*}

Solving for the particular solution. We have \(y(0)=2\) and \(y^{\prime}(0)=-1\text{.}\) Differentiating \(y(t)=c_{1}e^{-2t}+c_{2}e^{-3t}\) we get \(y^{\prime}(t)=-2c_{1}e^{-2t}-3c_{2}e^{-3t}\) and set up the following system:

\begin{align*} c_{1}+c_{2} \amp = \amp 2\\ -2c_{1}-3c_{2} \amp = \amp -1 \end{align*}

and get \(c_{1}=5,c_{2}=-3\text{.}\) So the particular solution is

\begin{equation*} y(t)=5e^{-2t}-3e^{-3t}. \end{equation*}
Example 3.2.3.

Let's find the general solution of

\begin{equation*} 2\frac{d^{2}y}{dt^{2}}+7\frac{dy}{dt}-4y=0. \end{equation*}
  1. We'll guess that the solution to a solution is \(y(t)=e^{rt}\) for some \(r\text{.}\) Then get
    \begin{equation*} \left(2r^{2}+7r-4\right)e^{rt}=0 \end{equation*}
    so that we must have \(2r^{2}+7r-4=\left(2r-1\right)\left(r+4\right)=0\) so that \(r=\frac{1}{2},-4\text{.}\)
  2. So \(y_{1}(t)=e^{t/2}\) and \(y_{2}(t)=e^{-4t}\) are solutions and
    \begin{equation*} y(t)=c_{1}e^{t/2}+c_{2}e^{-4t} \end{equation*}
    is the general solution.

Subsection 3.2.2 Derivatives as linear operators and the characteristic equation

It might seem unsatisfying to assume the form of an answer and then show that our guess worked. Is there a mathematical justification for making this guess? The answer is yes, if we're willing to push our understanding of differentiation slightly.

Define the differential operator \(D\) to be the operation of applying \(\frac{d}{dx}\) to a function \(y(x)\text{.}\) The operator \(D\) can be thought of as a function on functions:

\begin{equation*} D(\text{function}) = \text{ derivative}. \end{equation*}

Any order of derivative can be thought of this way. For example,

\begin{equation*} y''' = \frac{d^3 y}{dx^3} = \frac{d^3}{dx^3} y = D^3 y. \end{equation*}

We need a notion of \(D^0\) - what should that mean? It makes sense to set \(D^0(y) = y\) - that is, take no derivatives of \(y\text{.}\) We will use the symbol \(1\) to represent this function, the identity function that leaves \(y\) unchanged.

In Calculus, we learn that derivatives follow certain rules. Two of the most important are

\begin{equation*} D(f + g) = Df + Dg \end{equation*}

and

\begin{equation*} D(cf) = cD(f). \end{equation*}

These two properties together show that \(D\) is a linear function.

Using differential operator notation allows us to transform an ODE and use algebraic techniques to solve it using first order methods. Consider the equation

\begin{equation*} y'' + 3 y ' + 2 y = 0. \end{equation*}

This can be written

\begin{equation*} D^2 y + 3 D y + 2 y = 0 \end{equation*}

using operator notation. But since each operator is applied to the same input, \(y\text{,}\) we can use function notation to write

\begin{equation*} (D^2 + 3D + 2)(y) = 0. \end{equation*}

The operator expression \(D^2 + 3D + 2\) can be factored as symbols into \((D + 2)(D + 1)\) (that this is equal to the original equation applied to \(y\) is a consequence of linearity). Now, using the fact that the derivative of 0 is 0, we assert that the solutions to the equation

\begin{equation*} (D + 1)(D + 2)y = (D + 2)(D+1)y = 0 \end{equation*}

are the solutions to \((D + 1)(y) = 0\) and \((D + 2)y = 0\text{,}\) which are just the first order equations

\begin{gather*} y' + y = 0\\ y' + 2y = 0 \end{gather*}

These are separable, with solutions \(y_1 = c_1 e^{-x}\) and \(y_2 = c_2 e^{-2x}\text{.}\) So the general solution (justified by the principle of superposition, covered in the next section) to the ODE is

\begin{equation*} y = c_1 e^{-x} + c_2 e^{-2x}. \end{equation*}

Note that the operator polynomial \(D^2 + 3D + 2\) is precisely equivalent to the characteristic polynomial \(r^2 + 3r + 2\text{.}\)
